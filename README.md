# Find Your Spiritual Tiki Drink

An interactive personality quiz that matches you to one of 50+ classic and contemporary tiki cocktails based on your answers to 15 randomized questions.

**[Try it live](https://benswork-space.github.io/tiki-drink/)**

## How It Works

Answer 15 personality questions drawn from categories like Soul, Fruits, Elixirs, Temple, Island, and Voyage. Your answers map onto five personality axes:

- **Sweet vs. Boozy**
- **Serious vs. Laid Back**
- **Classic vs. Avant-Garde**
- **Simple vs. Complex**
- **Tropical vs. Spiced**

A Euclidean distance algorithm matches your personality profile against a database of 50+ tiki drinks — spanning foundational classics from the Don the Beachcomber era through modern revival cocktails — to find your best match plus two runner-ups.

## Features

- 50+ drinks with personality descriptions, taglines, and recipe highlights
- Animated personality profile visualization
- Share results via clipboard or X/Twitter
- Ambient tropical music synthesized with Web Audio API
- Deployed to GitHub Pages via GitHub Actions
